# Corne Keyboard

## Install QMK

[install QMK](https://docs.qmk.fm/newbs_getting_started)

## Keymap

```bash
qmk new-keymap -kb crkbd/rev4_1/standard -km <keymap>
```

## Compile 

```bash
qmk compile -kb crkbd/rev4_1/standard -km <keymap>
```

## Flash

```bash
qmk flash -kb crkbd/rev4_1/standard -km <keymap>
```
